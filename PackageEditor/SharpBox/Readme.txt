SharpBox - API for cloud based storage systems
----------------------------------------------

 
Copyright (c) 2010 Dirk Eisenberg, Yury Averkiev, Renald Dubus, Jay Ongg 
 

The SharpBox library supports the access to different cloud based storage 
systems by the .NET Framework. The library as self was written in C# and
should be compatibly with all .NET languages.

SharpBox is using a couple of third party products to support different web 
protocols. This distribution contains all needed 3rd party library and tools.
For additional information please visit the following websites of the 
authors:

- JSON.NET - http://json.codeplex.com/
- DevDefined OAuth - http://code.google.com/p/devdefined-tools/wiki/OAuth
